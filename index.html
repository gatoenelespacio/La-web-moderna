La Web Moderna

HTML

HTML quiere decir HyperText Markup language, lo cual significa "lenguaje de marcas de hipertexto", hace referencia al lenguaje de marcado utilizado en la creacón de páginas web.

CSS

CSS conocido en español como hoja de estilos en cascada, es una poderosa herramienta que se integra con HTML para proporcionar a los desarrolladores y usuarios un control avanzado sobre la presentación visual de las páginas web.

javaScript

Javascript es un lenguaje de programación que te permite implementar funciones complejas en páginas web, crear contenido de actualización dinámica, controlar multimedia, animar imágenes, etc.

HTTPS

HTTPS tambien conocido como "Protocolo de transferencia de Hypertexto. Se trata del protocolo de comunicación empleada para transferir informaciónn a través de la World Wide Web.
